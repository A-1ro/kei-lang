## お題

商品(コード+数量)リストとしきい値。数量がしきい値以下の商品の件数と数量合計を返す。しきい値 >= 0 を requires、結果の非負を ensures で明示。反例探索モードも試すこと。

## 結果: 9/10 — M31 キャプチャ一発 + generative が実バグを検出

```kei
func summarizeReorderCandidates(products: List<Product>, threshold: Int) -> ReorderSummary
  requires threshold >= 0
  requires products.all(p => p.quantity >= 0)
  ensures result.count >= 0
  ensures result.totalQuantity >= 0
{
  let candidates = products.filter(p => p.quantity <= threshold)
  return ReorderSummary { count: candidates.length, totalQuantity: candidates.fold(0, (acc, p) => acc + p.quantity) }
}
```
