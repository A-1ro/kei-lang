## 結果: 9/10 — 初回ドラフト一発エラーゼロ・修正ゼロ

```kei
extern Auth.verifyToken(token: String) -> Option<Int> uses Network.Read, Async
extern Users.fetchUser(id: Int) -> Option<UserInfo> uses Network.Read, Async

func getUserByToken(token: String, now: Int) -> Result<UserInfo, LookupError>
  uses Network.Read, Async
  requires now >= 0
{
  let userId = Auth.verifyToken(token) else fail LookupError.InvalidToken
  let user = Users.fetchUser(userId) else fail LookupError.UserNotFound(userId)
  return Ok(user)
}
```
