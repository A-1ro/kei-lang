## お題

小さな TODO API 1 本を Kei で書き、Cloudflare Workers にデプロイ可能状態まで持っていく。

- `GET /todos/:id` の 1 エンドポイント
- TS 側で固定 3 件の TODO ストアを組み立て、Kei ハンドラに closure DI で注入
- 見つからなければ 404、見つかれば 200 で `{id,title,done}` を返す
- Kei 関数に意味のある `requires`/`ensures` を最低 1 つ追加(テンプレ丸コピは不可)
- 完了パイプライン: `kei check --json` / `kei build` / `tsc --noEmit` /
  `wrangler deploy --dry-run` の全ステップ exit 0
