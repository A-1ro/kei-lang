## 結果: 初見一発でエラーゼロ(9/10)

```kei
module catalog.lookup

enum SearchResult {
  Invalid
  Found { message: String }
  NotFound { message: String }
}

func searchProductCode(codes: List<String>, query: String) -> SearchResult {
  let parsed = query.toInt()
  if parsed.isNone {
    return SearchResult.Invalid
  }
  if codes.contains(query) {
    return SearchResult.Found { message: "FOUND: " + query }
  }
  return SearchResult.NotFound { message: "NOT FOUND (" + codes.length.toString() + " items)" }
}
```

- `kei check --json` → diagnostics []、`kei fmt --check` → 一発正規形
- **M29/M30 の重点確認: `toInt()`(Option)・`contains`・`+` 連結・`length.toString()` を全て初見で正しく使用**。SKILL.md + spec v0.1 のみで完結(v0.2/v0.3 spec・examples 未参照)
