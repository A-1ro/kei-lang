## お題

会員制サービスの入会審査。年齢と延滞回数を受け取り、「18 歳以上 65 歳未満、かつ延滞回数 3 回未満」で入会可。可なら会費(30 歳未満 3000 / それ以外 5000)、不可なら理由付きで返す。

## 結果: 初見一発でエラーゼロ(9/10)

```kei
module membership.screening

enum MembershipResult {
  Approved(Int)
  Rejected { reason: String }
}

func screenMembership(age: Int, delinquencies: Int) -> MembershipResult
  requires age >= 0
  requires delinquencies >= 0
{
  let eligible = age >= 18 && age < 65 && delinquencies < 3
  if eligible {
    if age < 30 {
      return MembershipResult.Approved(3000)
    }
    return MembershipResult.Approved(5000)
  }
  return MembershipResult.Rejected { reason: "age or delinquency count does not meet membership criteria" }
}
```

- `kei check --json` → `diagnostics: []`。requires 2 件は `runtime` レベル報告。
- `kei fmt --check` → 無出力(初稿が正規形)。
- **M28 の重点確認: `&&` の 3 連結を初見で正しく書けた。** 左結合・比較との優先順位で混乱なし。
