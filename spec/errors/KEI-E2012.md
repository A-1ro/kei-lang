# KEI-E2012: 'Map.empty()' needs a type annotation

型検査: `Map<K, V>` の構築は `Map.empty()` の 1 形のみで、型引数を持たない(spec
kei-spec-v0.3-collections.md §7.3)。`K` / `V` は周辺の**期待型**(`let` の型注釈・関数呼び出しの
引数位置・関数の戻り値位置)から決定する。どの期待型も無い位置で `Map.empty()` を書くと型を
決定できず、このエラーになる。

```kei
let m = Map.empty()
// error: 'Map.empty()' requires a type annotation to determine its key/value types
```

## 修正

型注釈を加える、または期待型が伝播する位置(引数・戻り値)に置く。

```kei
let m: Map<String, Int> = Map.empty()

func emptyStock() -> Map<String, Int> {
  return Map.empty()   // OK: 戻り値位置の期待型から Map<String, Int>
}
```
